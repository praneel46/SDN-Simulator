# 🌐 SDN Network Simulator

An interactive web-based simulator demonstrating intelligent routing using **Dijkstra** and **A*** algorithms.
This project visualizes how Software Defined Networking (SDN) can dynamically compute optimal paths and respond to network changes in real time.

---

## 🚀 Features

* 🔷 **Interactive Network Graph**

  * Visual representation of nodes and links
  * Real-time path highlighting

* ⚡ **Algorithm Switching**

  * Toggle between **Dijkstra** and **A*** instantly
  * Compare behavior on the same topology

* 📦 **Packet Animation**

  * Smooth packet flow from source to destination
  * Adjustable speed control

* 💥 **Dynamic Link Failure**

  * Simulate network failures
  * Observe automatic path recalculation

* 📊 **Performance Comparison Dashboard**

  * Compare **Time, Cost, and Hops**
  * Visual chart using Chart.js

* 🧠 **Smart Analysis System**

  * AI-style analysis flow
  * Winner announcement with glow + sparkles
  * Explanation card describing why A* performs better

---
